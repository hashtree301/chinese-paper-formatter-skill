# 中文论文格式整理 (Chinese Paper Formatter)

本 Skill 用于自动化整理并不规范的 Word 论文内容。它依赖本机的 Microsoft Word 应用程序来原生应用中英文字体、边距缩进，并实现原生渲染目录更新。

## ⚠️ 重要的执行前置条件
由于本操作直接调用 COM 接口，这要求必须符合以下条件：
1. **已安装 Microsoft Word / Office**。(不支持 WPS 相关的直接 COM，必须是 MS Word)
2. 操作系统为 Windows。
3. Python 环境中已安装 `pywin32`。

## 如何使用此 Skill 解决用户的格式整理诉求：
1. **安装依赖**：向用户确认或通过脚本检查是否已经安装了 `pywin32`。如果没有，帮用户在终端执行 `pip install pywin32`。
2. **确认输入**：确认用户提供的待处理 `.docx` 文件路径（推荐使用绝对路径）。
3. **执行脚本**：运行此目录下自带的 `scripts/formatter.py` 脚本来进行自动化排版：
   ```bash
   python <当前skill绝对路径>/scripts/formatter.py "<输入文件路径>" "<输出文件路径>"
   ```
   *注意：如果未指定输出文件路径，脚本会将排版后的内容直接覆盖原文件。*
4. **人工提醒**：自动处理完成后，虽然绝大部分规范（如边距、标题、首行缩进、英文字体、目录）已被刷新，但请知会用户去手动检查一些复杂区域（例如跨页断行的复杂表格、不规范的公式等），避免出现自动化兼顾不到的极少数版面位移问题。
